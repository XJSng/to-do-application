Databases

What is a database?

A database is server for permanently storing data and work with that data in a meaningful way.

Creating our own database

There are multiple database systems:

  • MongoDB
  • MySQL
  • SQL Server
  • Oracle
  • MariaDB
  • PostgreSQL
  • CouchDB
  • etc.

They all have different philosophy in how they handle data but at the end of the day, all databases function to make it easy to work with data.

We are choosing MongoDB as it is beginner friendly and easy to use. We can sign up for a free account here. We will begin by building our first cluster.

  • We begin by choosing a cloud provider(AWS, Google, Azure)
  • Select our region (with free tier available)
  • Create cluster
  • Wait a while and your cluster would be created

CRUD Acronym & Our First Database

  2. Give it name of "TestDatabase.pets"

Read

Read can be replaced with retrieve from the database. The fact that we are viewing the object on MongoDB is part of the read acronym.
The filter feature is also another form of reading the database.

{species: "cat"} in the search tab

Update

Update simply means you can change or update an existing document object.
The pencil icon next to each object document allows users to edit any field like the "name" or "species" of the object.
